# ✨ Leadora CRM — Client Lead Management System

A full-stack Mini CRM system built for **Future Interns Full Stack Development Task 2 (2026)**.

Built by **Vani K Vasanthkumar**

---

## 🚀 Features

- 🔐 Secure Admin Login & Register (JWT Authentication)
- 📊 Dashboard with real-time stats & charts
- 👤 Lead Management (Add, Edit, Delete, Search, Filter)
- 📋 Pipeline View — track leads across stages
- 📈 Analytics — Bar, Pie & Line charts
- 📞 Contacts — view all leads as contacts
- 🌙 Beautiful Midnight Blue & Cyan dark theme

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Recharts, React Router |
| Backend | Node.js, Express.js |
| Database | MongoDB Atlas |
| Auth | JWT + Bcrypt |
